# SmartDoorSystem
Door Lock System\ 
Embedded System Project using using 2 atmega16 microcontrollers\
by implementing the following drivers\
1.GPIO\
2.LCD\
3.KEYPAD\
4.DCMotor\
5.UART\
6.EEPROM\
